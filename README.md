# Delego

Delego is a Flutter-based application aimed at simplifying task delegation and management. It allows users / Delegate to track, and manage tasks efficiently during MUN.

## Features

- **Task Assignment**: Easily delegate tasks to team members with due dates and priority levels.
- **Progress Tracking**: Monitor the status of assigned tasks in real-time.
- **Notifications**: Receive timely reminders about upcoming deadlines and task updates.
- **Cross-Platform Support**: Available on Android, iOS, macOS, Linux, and Windows platforms.

## Getting Started

To set up the development environment for Delego, follow these steps:

1. **Clone the repository**:

   ```bash
   git clone https://github.com/MithilSave/Delego.git
   cd Delego
   ```

2. **Update Java Development Kit (JDK)**:

   Ensure you have the latest version of JDK installed. You can download it from the [official Oracle website](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html).

3. **Install Flutter**:

   If you haven't installed Flutter yet, follow the official installation guide [here](https://docs.flutter.dev/get-started/install).

4. **Upgrade Flutter**:

   Make sure you're using the latest version of Flutter:

   ```bash
   flutter upgrade
   ```

5. **Install Dependencies**:

   Clean previous builds and fetch the necessary packages:

   ```bash
   flutter clean
   flutter pub get
   ```

6. **Run the Application**:

   Start your preferred emulator or connect a physical device, then execute:

   ```bash
   flutter run
   ```

   Note: The initial build might take some time as it downloads required packages.
